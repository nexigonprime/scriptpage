Documentação do Projeto

Introdução

Bem-vindo à documentação do projeto.

O projeto foi iniciado para lhe ajuda com scripts de automatização de instalação de paineis de jogos com no momento pterodactyl, jexactyl e pelican

O que é

Eles são uma plataforma de gerenciamento de servidores de jogos que facilita o processo de hospedagem, administração e controle de servidores de jogos de forma eficiente e segura. Ele é projetado para ajudar tanto os administradores de servidores quanto os usuários finais a gerenciar servidores de jogos com facilidade.
